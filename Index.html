ESP-32

El ESP32 es un microcontrolador (SoC) desarrollado por Espressif Systems que integra conectividad Wi-Fi y Bluetooth, lo que lo hace ideal para aplicaciones de Internet de las Cosas (IoT).

El ESP32 es muy versátil y se puede utilizar tanto como un sistema autónomo como un dispositivo esclavo conectado a un MCU principal. Es popular en proyectos de electrónica y desarrollo de prototipos debido a su alta integración y bajo costo.
  • Procesador: CPU Xtensa de doble núcleo de 32 bits LX6, operando a 160 o 240 MHz.
  • Memoria: 520 KB de SRAM y 448 KB de ROM.
  • Conectividad: Wi-Fi 802.11 b/g/n y Bluetooth v4.2 BR/EDR y BLE.
  • Periféricos: Incluye sensores táctiles capacitivos, sensores Hall, interfaz de tarjeta SD, Ethernet, SPI de alta velocidad, UART, I²S e I²C.

Arduino

El Arduino es una plataforma de electrónica de código abierto que facilita la creación de proyectos interactivos. Está basada en hardware y software fáciles de usar, lo que la hace accesible tanto para principiantes como para expertos.

  • Plataforma de código abierto: Tanto el hardware como el software de Arduino son de código abierto, lo que permite a los usuarios modificar y mejorar sus proyectos.
  • Facilidad de uso: Diseñado para ser accesible tanto para principiantes como para expertos, con una gran cantidad de recursos y documentación disponibles.
  • Variedad de placas: Existen diferentes modelos de placas Arduino, como Arduino Uno, Arduino Mega y Arduino Nano, cada una con diferentes capacidades y características.
  • Lenguaje de programación: Utiliza una variante de C/C++ y se programa a través del entorno de desarrollo integrado (IDE) de Arduino.

ESP-8266

El ESP8266 es un microcontrolador de bajo costo con conectividad Wi-Fi, desarrollado por Espressif Systems. Es muy popular en proyectos de Internet de las Cosas (IoT) debido a su capacidad para conectarse a redes Wi-Fi y su facilidad de programación.

  • Procesador: CPU RISC de 32 bits Tensilica Xtensa LX106, operando a 80 MHz (con posibilidad de overclocking a 160 MHz).
  • Memoria: 64 KB de RAM de instrucciones, 96 KB de RAM de datos y soporte para memoria flash externa de hasta 16 MB.
  • Conectividad: Wi-Fi 802.11 b/g/n a 2.4 GHz, compatible con WPA/WPA2.
  • Periféricos: 16 pines GPIO, PWM en todos los pines (10 bits), ADC de 10 bits, UART, SPI, I²C e I²S.
